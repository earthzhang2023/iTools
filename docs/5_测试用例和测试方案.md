# iTools - 测试用例和测试方案文档

## 1. 文档概述

### 1.1 文档目的
本文档描述 iTools 纯前端在线工具网站的测试策略、测试方案、测试用例和验收标准，确保产品质量符合预期。

### 1.2 测试范围
- 功能测试
- 性能测试
- 兼容性测试
- 可用性测试
- 响应式测试
- 安全测试

### 1.3 测试策略
采用分层测试策略：
- **单元测试**：工具函数、组件
- **集成测试**：模块间交互、完整流程
- **E2E 测试**：用户关键路径
- **手动测试**：UI/UX、视觉验证

```typescript
// tools/base64/__tests__/index.test.ts
import { describe, it, expect } from 'vitest'
import { base64Encode, base64Decode } from '../index'

describe('Base64', () => {
  it('should encode string to base64', () => {
    expect(base64Encode('Hello')).toBe('SGVsbG8=')
  })
  
  it('should decode base64 to string', () => {
    expect(base64Decode('SGVsbG8=')).toBe('Hello')
  })
  
  it('should handle empty string', () => {
    expect(base64Encode('')).toBe('')
    expect(base64Decode('')).toBe('')
  })
  
  it('should throw error on invalid input', () => {
    expect(() => base64Decode('invalid!')).toThrow()
  })
})
```

### 8.2 组件测试

#### 测试示例
```typescript
// components/common/__tests__/CopyButton.test.ts
import { describe, it, expect, vi } from 'vitest'
import { mount } from '@vue/test-utils'
import CopyButton from '../CopyButton.vue'

describe('CopyButton', () => {
  it('should render correctly', () => {
    const wrapper = mount(CopyButton, {
      props: { text: 'test' }
    })
    expect(wrapper.text()).toContain('复制')
  })
  
  it('should copy text on click', async () => {
    vi.spyOn(navigator.clipboard, 'writeText').mockResolvedValue(undefined)
    
    const wrapper = mount(CopyButton, {
      props: { text: 'test' }
    })
    
    await wrapper.trigger('click')
    expect(navigator.clipboard.writeText).toHaveBeenCalledWith('test')
  })
  
  it('should emit copy event on success', async () => {
    vi.spyOn(navigator.clipboard, 'writeText').mockResolvedValue(undefined)
    
    const wrapper = mount(CopyButton, {
      props: { text: 'test' }
    })
    
    await wrapper.trigger('click')
    expect(wrapper.emitted('copy')).toBeTruthy()
  })
})
```

```typescript
// e2e/home.spec.ts
import { test, expect } from '@playwright/test'

test.describe('Home Page', () => {

    await expect(page).toHaveTitle(/iTools/)

  })
})
```
